# College Event Manager

A web-based application designed to streamline the process of planning, managing, and organizing events within a college or organization. This project provides an intuitive interface for administrators, organizers, and participants to coordinate schedules, venues, and registrations seamlessly.

---

## 🚀 Features

* **Event Dashboard:** A centralized hub to view upcoming, ongoing, and completed events.
* **User Management:** Separate roles and views for Administrators, Event Organizers, and Attendees.
* **Registration System:** Simple and secure registration form for participants.
* **Responsive Design:** Clean, modern user interface optimized for both desktop and mobile viewing.

---

## 🛠️ Tech Stack

The project is built using a lightweight and fast web stack:

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla/ES6)
* **Template Views:** Dynamic rendering structure using the `views/` directory.
* **Assets:** Static resources (images, styles, client-side scripts) managed inside the `public/` folder.
